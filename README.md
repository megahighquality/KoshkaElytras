
KoshkaElytras — datapack для кастомных элитр (обновлён под Minecraft 1.21.10)

Установка:
1. Поместите zip в папку world/datapacks/ или распакуйте содержимое архива в отдельную папку внутри datapacks.
2. Перезагрузите мир (/reload) или сервер.
3. Проверьте лог сервера — при успешной загрузке будет сообщение о загруженных datapack.

Важно:
- Если datapack использует asset_id (custom_elytra:...), для корректного отображения элитр клиентам требуется соответствующий resource pack.
- Для автоподключения resource pack сервер может указать URL в server.properties (resource-pack и resource-pack-sha1).
- Проверьте, что папка с advancement называется "advancements" (во множественном числе).
